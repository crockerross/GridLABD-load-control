Source Code edits were by Stephanie Crocker Ross at the University of Michigan, on or before 9/7/2018.

The files within "GridLABD_SourceCode" are the source code files that were edited. Unedited source code files are not included in the repository. 
All source code files used in this project are from the 3.1.0-5176 64 bit build version of GridLAB-D. 

The key changes made to the source code are in market\passive_controller.cpp and in residential\house_e.cpp 
These changes enable closed loop control (turning HVACs on or off) via a probabilistic command.

Build version of gridlabd is 3.1.0-5176 64 bit.

MATLAB version is 2014b 64bit.
